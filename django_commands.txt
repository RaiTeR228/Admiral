Поля бд
текстовые поля
    CharField - строка с ограничением длины - name = models.CharField(max_length=100)
    TextField - длинны текст без ограничения - description = models.TextField()
    EmailField - Email с валидацией - email = models.EmailField 

price = models.DecimalField(max_digits=10, decimal_places=2, default=0.00) # max_digits - максимальное количество цифр, decimal_places - количество цифр после запятой, default - значение по умолчанию

числовые поля
    IntegerField - целое число - age = models.IntegerField()
    FloatField - Число с плавающей точкой - weight = models.FloatField()

Дата и время
    DateField - дата(без времени) - data = models.DateField()
    DateTimeField - дата и время - create_at = models.DateTimeField(auto_now_add=True)
    TimeField - время без даты - start_time = models.TimeField()

Логические поля
    BooleanField = True/False - is_activate = models.BooleanField(default=True)

Спец поля 
    UUIDField - уник. индефр. - id = models.UUIDField(default=uuid.uuid64, editable=Fasle)

флаги
    null=True - разрешает базе данных хранить NULL (пустое значение) в этом поле
    blank=True - разрешает Django валидации пропускать это поле при заполнении форм (не требовать значение)

fail2ban

https://secureflow.ru/guides/fail2ban-nastroyka-zashchita-ssh
Пример /etc/fail2ban/jail.local:

[sshd]
enabled = true
port = ssh
maxretry = 2
bantime = 10m
findtime = 5m
Пояснение параметров:

enabled — включить фильтр
port — порт SSH (можно указать нестандартный)
maxretry — сколько неудачных попыток допускается
findtime — за какой период анализируются попытки
bantime — время блокировки IP

КОМПИЛЯЦИЯ  APK файла в Android studio

вводим команду в терминале(Alt+F12): 
    npx expo prebuild

Это создаст в корне проекта папку android со всеми необходимыми нативными файлами.
Потом переходим в папку android
    cd android
и вводим команду для компиляции apk в дебаг режиме
    ./gradlew assembleDebug
или в релиз
    ./gradlew assembleRelease

очистка кеша компилятора
	./gradlew clean

создание проекта 
	npx create-expo-app <name project>

Запуск проекта через gunicorn
    gunicorn --bind 0.0.0.0:8000 папка проекта где settings.py и wsgi .wsgi.application

для автоматизации запуска сервера, чтобы работал всегда
