Flexbox сетка для контента
по умолчанию весь контент в колонку
flexDirection:"column" - контент в колонку
flexDirection:"row" - контент в строку

выравнивание элементов по главной оси
justufyContent:'center'
justufyContent:'flex-start' - в начало
justufyContent:'flex-end' - в конец
justufyContent:'space-between' - растояние максимальное между элементами
justufyContent:'space-around' - растояние вокруг
justufyContent:'space-evenly' - равномерное растояние

выравнивание элементов по поперечной оси
alignItems:'center' - допустим кнопка отцентруется у экрана пользователя

само свойство flex у элемента
flex:1 - означает занять все доступное место
например у главное контейнера в котором другие элеименты можно поставить
flex:1
если не поставить flex:1 контейнер может занимать высоту своего содержимового
