    npx create-expo-app@latest название(client)

node_modules - не переносим для сдачи или других проект
чтобы подтянуть зависимости которые устанавливали перед запуском подтянуть
    npm i
    npx i

npm - менеджер пакетов
npx - запускает пакеты без постоянной установки

create-expo-app генератор react native проект на expo
expo - для быстрых приложений, интерфейсов, игр, прототипов
клиентских приложений

react native cli без expo - когда нужен полный контроль над android/ios
с нативной частью свои srift - модули java kotlin сложный sdk
специфичная интеграция

и телефон и пк должны находится в одной сети wifi

для запуска перейти в директорию для запуска
    npm run dev
    npx expo start

основная папка сбора app/
index - главный экран
explore - второй экран
_layout - общий для всех навигация

папка components/
    сюда складываем свои переиспользуемые компаненты *.tsx
    все tsx что создаем компаненты всегда с большой буквы

как создавать стилистику для тегов(объектов)
создается отдельная константа const style = StyleSheet.create({
    объект:{ключ стиля: значение}
})

в самом теге <Text style = {style.hero}>писька</Text>  

когда создали компанент нам нужно вывести его, либо в отдельный большой блок если он явл дочерним
либо выввести его сразу на экран на прямую
мы идем в index.tsx это явл нашей точкой сбора компонентов
